# UK Study Work Sponsorship Analysis Dashboard
This project analyzes UK study and work visa sponsorship trends using data from the UK Government Open Data platform. The goal of the project is to transform raw migration data into meaningful insights through SQL data processing and interactive Power BI visualizations.

The datasets were first imported into Microsoft SQL Server, where data cleaning, transformation, and merging were performed using T-SQL queries. The cleaned and merged datasets were then connected to Power BI to create an interactive analytical dashboard.

The dashboard explores migration patterns from 2010 to 2023, highlighting trends in visa applications, nationality distributions, regional sponsorship activity, and extension rates. Advanced DAX calculations and multiple Power BI visualizations were used to analyze and present the data effectively.

Key Features


•	Data cleaning and transformation using SQL Server

•	Dataset merging using T-SQL

•	Data modeling and analysis using Power BI

•	Interactive visualizations with advanced DAX measures

•	Analysis of study and work visa sponsorship trends


Dashboard Insights


The dashboard provides several analytical views including:

•	Total study and work applications by year

•	Year-over-year growth analysis by quarter

•	Top nationalities applying for study and work visas

•	Regional distribution of repeat sponsorship applications

•	Extension rate analysis for study and work visas

•	Applications by institution type and industry sector

These visualizations help identify migration patterns, seasonal trends, and key countries contributing to UK sponsorship programs.


Tools & Technologies
